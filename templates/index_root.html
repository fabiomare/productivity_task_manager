Welcome to the To-Do API!

Help:

  • To create a new to-do item, make a POST request to 127.0.0.1:8000/todos/.
  • To retrieve all to-do items, visit 127.0.0.1:8000/todos/ or make a GET request.
  • To see an interactive documentation page, visit 127.0.0.1:8000/docs.
  • To see the ReDoc documentation, visit 127.0.0.1:8000/redoc.

Thank you for using the To-Do API!
